Документация report_maker:

Проект создан для упрощения создания отчётов для математики, программирования(дневник программиста) и других дисциплин

Для запуска нужно запустить "run.exe"
Далее нужно обязательно выбрать рабочую директорию(папку)
После выбора нужно определить тип создания
В конце нажать кнопку пуска
При возникновении ошибок, написать в вк vk.com/chingiz.khabalov и скинуть log.txt из папки /logs

Системные требования:
Windows 7 или 10(тесты других версий не проводились).
Экран с разрешением не менее 800x600.
Место на диске: 128 Мб.

Требования для фотографий на обработку:
Тип файла: ".jpg", ".png", ".jpeg" и ".bmp"
Разрешение картинки в пределах "720x1018 - 3000x4000"
Соотношение сторон в не более 2:1(к обеим сторонам)
